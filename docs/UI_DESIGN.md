## Animations

### Page Load
```
1. Header: Sticky position, always visible
2. Hero: Fade in from top (0.8s)
3. Form: Slide up from bottom (0.6s)
4. Features: Fade in (1s, delayed 0.3s)
```

### Interactions
```
Buttons:
- Hover: Lift up 2px, increase shadow
- Active: Return to original position
- Disabled: 70% opacity, no pointer

Inputs:
- Focus: Border color change, shadow glow
- Error: Red border, shake animation

Cards:
- Hover: Lift up 8px, increase shadow
```

### Loading State
```
Button shows:
┌──────────────────────┐
│ ⟳ Predicting...     │
└──────────────────────┘

Spinner: Rotating circle animation
```

## Responsive Breakpoints

### Desktop (> 768px)
```
- Full 3-column layout for features
- Form: 600px max width, centered
- Generous padding and spacing
```

### Tablet (640px - 768px)
```
- 2-column layout for features
- Form: Full width with padding
- Adjusted font sizes
```

### Mobile (< 640px)
```
- Single column layout
- Stacked form inputs
- Reduced padding
- Smaller font sizes
- Touch-friendly buttons (min 44px)
```

## Accessibility Features

✅ **Semantic HTML**: Proper heading hierarchy (h1, h2, h3)
✅ **ARIA Labels**: Screen reader support
✅ **Keyboard Navigation**: All interactive elements accessible
✅ **Focus Indicators**: Visible focus states
✅ **Color Contrast**: WCAG AA compliant
✅ **Error Messages**: Clear, descriptive validation errors
✅ **Loading States**: User feedback during async operations

✅ Chrome/Edge 90+

✅ Mobile browsers (iOS Safari, Chrome Mobile)

## Performance Optimizations

✅ **CSS**: Minimal, no frameworks (lightweight)
✅ **Images**: SVG icons (scalable, small)
✅ **Animations**: GPU-accelerated (transform, opacity)
✅ **Lazy Loading**: Components load on demand
✅ **Code Splitting**: Vite handles automatically
✅ **Caching**: Browser caching for static assets
